Project Overview

CardHub – Smart Invitation & Template Builder

CardHub is a full-stack web application inspired by modern design platforms. It allows users to browse invitation templates, preview community reviews, customise fonts, colours and backgrounds, then download a share-ready invite.

Developed by

Amit Singh & Naman Chausriya

Created as an academic and portfolio-ready web application to demonstrate practical knowledge of frontend UI design, backend APIs and database integration.

Tech Stack Used

  • Frontend: HTML5, CSS, modern UI components.
  • Backend: Python Flask for routing, templates and sessions.
  • Database: SQLite with Flask-SQLAlchemy ORM.
  • Export: html2canvas to capture preview and download PNG.
  • UX Extras: hover animations and mobile-friendly interactions.

What Users Can Do

  • Browse templates by category.
  • Open preview with comments, likes and ratings.
  • Enter live editor to customise design.

Inside the Editor

  • Change title, lines and label.
  • Choose font family and size.
  • Select background colour or upload image.
  • Download high-quality PNG.

Social Features

  • Users can register and save customised cards.
  • Each template has review & rating system.
  • Discover page shows trending and most-liked designs.
